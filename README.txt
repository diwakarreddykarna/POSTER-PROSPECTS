NOTE: Install Python

----Install needed libraries----
1. pip install opencv-python
2. pip install pytesseract
3. pip install webcolors
4. pip install googletrans==4.0.0-rc1
5. Download and configure Tesseract OCR as described in this 
* Download tesseract exe from https://github.com/UB-Mannheim/tesseract/wiki.
* Install this exe in C:\Program Files (x86)\Tesseract-OCR
* Open virtual machine command prompt in windows or anaconda prompt.

----Running the application------
To run the application, type python manage.py runserver in command prompt/terminal.
